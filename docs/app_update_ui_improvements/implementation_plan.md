# アプリUIおよび機能の改善（マニュアル拡充の代替）

A4マニュアルに情報を詰め込む代わりに、アプリのプログラムやUIを改修してユーザービリティを向上させます。

## Proposed Changes

### 1. `src/components/Scanner.tsx` (ライト点滅バグの修正)
- **現状**: カメラのマージを停止（`stopCamera`）する際、メディアトラックをいきなり `stop()` しているため、OSやブラウザによっては「ライトがついた状態」が保持されてしまい、次回のカメラ起動時に意図せず再点灯してしまいます。
- **改修**: カメラを起動する際（`startCamera`）、前回ライトをオンにした記憶が残っていても、常に `torch: false` から安全にスタート（初期化）するように初期化ロジックを追加し、バグを防ぎます。

### 2. `src/components/Generator.tsx` (生成機能UIの小ヒント追加)
- **手動入力（複数）モード**のテキストやプレースホルダーを少し親切に変更します。「改行で区切って複数入力できます」など、初めて使う人でも直感的に分かるように文言を微調整。

### 3. `src/App.tsx` & `src/components/HelpMenu.tsx` (使い方・Tips画面の新規追加)
- [NEW]コンポーネントとして `HelpMenu.tsx` またはそれに準ずるモーダルを作成し、`App.tsx` のヘッダー部分（右肩など）に「❓ 使い方・Tips」ボタンを配置します。
- この画面に、ご要望のあった以下の細かい説明を記載し、いつでもアプリ内から確認できるようにします。
  - **出力について**: 「一覧タブからTSV出力が可能です。JANの0落ちを防ぐためTSV出力を推奨します。用途に応じて出力列のカスタマイズや、『個数を行展開』（1個につき1行にする）を利用してください」
  - **リスト管理**: 「作成から7日以上経過したリストには『7日超』というラベルが付きます。古いデータは随時整理をお願いします」
  - **カメラ機能**: 「カメラが起動しない場合は、ブラウザの設定でカメラへのアクセスを許可してください」
  - **相場検索**: 「商品名に『Amazon』等と入れておくことで、ワンタップで相場検索が可能です」
